*mli-cycler.txt* OCaml Implementation & Interface File Cycling	   *mli-cycler*

==============================================================================

This plugin cycles between `foo.ml`, `foo.mli`, and `foo_intf.ml` files.

:MliCyclerNext						*:MliCyclerNext*
		Go to the next file in the cycle.

:MliCyclerPrev						*:MliCyclerPrev*
		Go to the previous file in the cycle.

:MliCyclerList						*:MliCyclerList*
		Display the list of available files in the cycle.

:MliCyclerListFzf					*:MliCyclerListFzf*
		Populate an |fzf| buffer with the available files in the cycle
		to navigate between them.
